# 维宝宁销售话术对练 - 快速上手指南

## 🚀 3 步开始练习

### 1️⃣ 添加机器人
在飞书搜索：**维宝宁** → 添加为好友

### 2️⃣ 开始练习
发送：**开始练习**

### 3️⃣ 选择医生
回复数字：**1-5**（建议从 3 开始）

---

## 📱 常用指令

| 发送 | 功能 |
|------|------|
| `开始练习` | 开始新对练 |
| `帮助` | 查看指令列表 |
| `结束` | 结束当前对练 |
| `查看报告` | 学习报告 |

---

## 💡 练习建议

- 每天练习 **15-20 分钟**
- 从 **主治医师**（难度⭐⭐⭐）开始
- 准备 **开场白** + **3个核心卖点**
- 多用 **异议处理** 话术
