# 销售话术库

## 标准拜访流程话术

### 1. 开场白（Opening）

**目标**：建立关系、引起兴趣、获得时间许可

#### 模板1：直接切入型（适合时间紧张的医生）
```
"X主任您好，我是XX公司的医药代表小王。

今天想占用您2分钟时间，介绍一款在[适应症领域]
有突破性进展的新产品——维宝宁。

它刚刚被[指南名称]推荐为一线用药，
我想您可能会感兴趣。"
```

#### 模板2：学术引导型（适合学术型医生）
```
"X主任，上次您提到的[相关学术话题]，
我最近看到一个很有意思的研究。

是关于维宝宁在[适应症]的最新临床数据，
发表在[期刊名称]上，想和您分享一下。"
```

#### 模板3：关系维护型（适合老客户）
```
"X主任，好久不见！最近科室工作忙吧？

上次您提到的[上次话题]，我回去查了一下资料，
这次给您带过来了。顺便想跟您更新一下维宝宁的最新信息。"
```

---

### 2. 探询需求（Probing）

**目标**：了解患者类型、用药习惯、未满足需求

#### SPIN提问技巧

**S - Situation（现状）**
- "您科室目前[适应症]的患者大概是什么年龄段居多？"
- "这类患者目前主要用什么方案治疗？"

**P - Problem（问题）**
- "目前的治疗方案，患者反馈最多的问题是什么？"
- "有没有遇到[具体问题]的情况？"

**I - Implication（暗示）**
- "这些问题如果长期得不到解决，对患者会有什么影响？"
- "您觉得目前的方案还有哪些提升空间？"

**N - Need-payoff（需求-效益）**
- "如果有一款产品能解决[具体问题]，您会考虑尝试吗？"
- "更快的起效速度，对这类患者意味着什么？"

---

### 3. 产品介绍（Presentation）

**目标**：清晰传递产品价值，使用FAB法则

#### FAB话术模板

**Feature（特性）→ Advantage（优势）→ Benefit（利益）**

```
"维宝宁采用[技术/成分特性]，
这使得它在[某方面]比传统药物更有优势，
对患者来说意味着[具体利益]。"
```

#### 核心卖点FAB示例

**卖点1：[待填写]**
```
F：维宝宁具有[特性]
A：相比传统药物，[优势]
B：对患者来说，[利益]
```

**卖点2：[待填写]**
```
F：维宝宁的[特性]
A：这使得[优势]
B：医生可以[利益]
```

**卖点3：[待填写]**
```
F：维宝宁采用[特性]
A：避免了[优势]
B：提高了[利益]
```

### 5. 促成成交（Closing）

**目标**：获得承诺、推动试用/处方

#### 成交技巧

**1. 直接请求法**
```
"X主任，维宝宁在您刚才提到的那类患者中效果特别好，
我建议您可以先开几盒试试看效果。"
```

**2. 二选一法**
```
"您看是先给门诊患者试用，还是住院患者先开始？"
```

**3. 从众心理法**
```
"XX医院的相关科室上个月已经开始用了，
反馈效果很不错。您要不要也试试看？"
```

**4. 试用推进法**
```
"我先给您放几份样品在这里，
遇到合适的患者您可以先试用，
效果好的话我们再谈进药的事。"
```

### 6. 跟进服务（Follow-up）

**目标**：建立长期关系、提供学术支持

```
"X主任，这是我的联系方式，有任何问题随时找我。

下个月我们有一个关于[学术话题]的学术会议，
我到时候把资料给您带过来。

感谢您的宝贵时间，再见！"
```

## 不同医生类型的沟通策略

### 学术型医生（主任级专家）
- **关注点**：循证医学、临床数据、指南推荐
- **话术特点**：
  - 引用权威文献
  - 强调临床研究数据
  - 讨论作用机制
- **示例**：
  ```
  "这项III期临床研究显示，维宝宁的疗效显著优于对照组
  （p<0.001），且安全性良好。"
  ```

### 务实型医生（科室主任）
- **关注点**：疗效、安全性、性价比
- **话术特点**：
  - 简洁明了
  - 强调实际效果
  - 提及成本效益
- **示例**：
  ```
  "维宝宁起效快，不良反应少，患者依从性好，
  能减少您的管理成本。"
  ```

### 经验型医生（主治医师）
- **关注点**：用药便利性、患者反馈
- **话术特点**：
  - 强调使用便利性
  - 分享其他医生经验
  - 提及患者满意度
- **示例**：
  ```
  "用了维宝宁的医生反馈，患者满意度明显提高，
  复诊率也提升了。"
  ```

### 学习型医生（住院医师）
- **关注点**：基础知识、适应症、用法用量
- **话术特点**：
  - 耐心讲解基础
  - 提供学习资料
  - 鼓励提问
- **示例**：
  ```
  "关于维宝宁的适应症，我详细给您讲一下...
  这里有一份详细的学习资料供您参考。"
  ```

## 常用金句库

### 建立信任
- "我跟很多像您这样的专家交流过..."
- "根据最新的临床数据显示..."
- "您的经验非常丰富，我想听听您的看法..."

### 强调优势
- "维宝宁的核心优势在于..."
- "这一点是其他产品做不到的..."
- "从临床数据来看..."

### 处理顾虑
- "您的顾虑很有道理..."
- "很多医生一开始也有类似的担心..."
- "让我用数据来解释一下..."

### 促成成交
- "要不我们先试试看？"
- "我相信您用了之后会满意的..."
- "下次我再来拜访，听听您的使用反馈..."

## 禁忌话术

❌ **不要做的事**
- 贬低竞品
- 过度承诺疗效
- 施压式推销
- 打断医生说话
- 占用过多时间

❌ **不要说的话**
- "我们的产品是最好的"
- "其他医生都在用"
- "不用会后悔的"
- "便宜没好货"
