# 异议处理话术库

## 异议处理黄金法则：APRC

```
A - Acknowledge (认同)
P - Probe (探询)
R - Respond (回应)
C - Confirm (确认)
```

---

## 类型1：价格异议

### 异议表现
- "太贵了"
- "比竞品贵不少"
- "医保不报吧？"
- "患者可能接受不了"

### 处理策略

**APRC话术模板**：

```
A（认同）：
"您说得对，维宝宁的价格确实比传统药物要高一些。
很多医生一开始也有这个顾虑。"

P（探询）：
"我想了解一下，您目前用的是什么方案？
患者的依从性怎么样？"

R（回应）：
"虽然单价高一些，但考虑到：
1. 疗效更确切，能减少并发症治疗费用
2. 安全性更好，减少了不良反应处理成本
3. 患者依从性好，长期看反而更经济

有研究显示，使用维宝宁的患者总体医疗成本降低了XX%。
而且，医保已经纳入了报销目录，患者自付比例并不高。"

C（确认）：
"您看，从长期疗效和总体成本来看，
维宝宁的性价比其实是很有优势的。
您要不要先给几个合适的患者试试看？"
```

### 关键数据准备
- 总体医疗成本对比
- 医保报销比例
- 长期疗效带来的成本节约
- 患者自付金额对比

## 通用原则

### ✅ 要做的
- 认真倾听，不打断
- 表示理解和认同
- 用数据说话
- 提供选择，不强迫
- 保持专业和礼貌

### ❌ 不要做
- 与医生争论
- 贬低竞品
- 过度承诺
- 表现出不耐烦
- 强推强卖
