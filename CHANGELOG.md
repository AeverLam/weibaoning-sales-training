# 维宝宁销售话术对练 - 更新日志

## v2.0 - AI智能版（2026-04-01）

### 🎉 重大更新：方案B已实现

**核心功能：**
- ✅ AI智能医生回复（基于LLM）
- ✅ 实时评估用户回答质量
- ✅ 个性化反馈和学习建议

### 新增文件
- `scripts/ai_dialogue_engine.py` - AI对话引擎
- `render_app_ai.py` - AI版飞书机器人
- `AI_DEPLOY_GUIDE.md` - AI版部署指南

### 技术架构
```
用户输入 → AI对话引擎 → LLM生成医生回复
                ↓
         LLM评估回答质量
                ↓
         生成个性化反馈
                ↓
         返回给用户
```

### 支持的LLM
- OpenAI GPT-4 / GPT-3.5
- Anthropic Claude 3
- Moonshot Kimi（推荐，国内可用）

## 升级建议

### 适合升级到v2.0的情况
- 对对话真实性要求高
- 有LLM API预算
- 希望获得更准确的评估
- 需要个性化学习建议

### 继续使用v1.0的情况
- 预算有限
- 对对话真实性要求不高
- 需要离线使用
- 数据安全要求极高
